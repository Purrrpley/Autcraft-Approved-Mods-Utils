<p>Mods are modifications to the Minecraft game that make it behave differently. Sometimes they're small, like a minor graphical change; sometimes they're big, like adding new blocks and items; and sometimes they're unfair, like x-ray. In an effort to ensure that Autcraft is a safe and fair place to play, where everyone has similar experiences and opportunities, only some mods are allowed. This is a list of all of the mods currently allowed to be used on the Autcraft server. If a mod you want to use is on this list, please read the entire section for that mod so that you can see if there are any parts of that mod that are disallowed. If there is a mod that you want to use that is not on this list, please see if it has been suggested before and if not, suggest it on <a rel="nofollow" target="_blank" class="external text" href="https://www.autcraft.com/forum/m/16338872/viewforum/3071549">the forums</a> and don't use it unless it is put on this list.
</p><p>Note: If a mod on this list requires another mod to work, that mod is also allowed.
</p>

<h2><span class="mw-headline" id="Mod_loaders">Mod loaders</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h2>
<p>Most mods use either the Fabric or Forge mod loader. To use a mod, you'll usually have to install the mod loader first.
</p>
<h3><span class="mw-headline" id="How_to_Install_Fabric">How to Install Fabric</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h3>
<ol><li> Download the Fabric installer from <a rel="nofollow" target="_blank" class="external text" href="https://fabricmc.net/use/">fabricmc.net/use</a>. Use the Windows/.EXE method if you are on Windows, or the universal method if you're on another platform (or don't like the SmartScreen warning). Note that to open the .jar file, you'll need to install Java first which you can do from <a rel="nofollow" target="_blank" class="external text" href="https://adoptopenjdk.net/?variant=openjdk16&amp;jvmVariant=hotspot">adoptopenjdk.net</a>.</li>
<li> Open the file that you downloaded. If done correctly, the installer window should appear.</li>
<li> Select the Minecraft version you want to use.</li>
<li> Click "Install"</li>
<li> Open the Minecraft launcher (closing it first if it was already open).</li>
<li> A new Fabric installation profile should appear. Select it, put the mods you want to use into a "mods" folder in your Minecraft directory (instructions on how to find it <a rel="nofollow" target="_blank" class="external text" href="https://minecraft.fandom.com/wiki/.minecraft">here</a>), and you're ready to play!</li></ol>
<h3><span class="mw-headline" id="How_to_Install_Forge">How to Install Forge</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h3>
<ol><li> Download the installer for the version of Forge for the Minecraft version you want to use from <a rel="nofollow" target="_blank" class="external text" href="https://files.minecraftforge.net/net/minecraftforge/forge/">files.minecraftforge.net</a>. Note that clicking on the download link will take you to an ad page before letting you download the file, do NOT click on anything on that page except for the "Skip" button at the top-right! If you did that correctly, you should get a file called something like `forge-&lt;minecraft_version&gt;-&lt;forge_version&gt;-installer.jar`.</li>
<li> To open the .jar file, you'll need to install Java first which you can do from <a rel="nofollow" target="_blank" class="external text" href="https://adoptopenjdk.net/?variant=openjdk16&amp;jvmVariant=hotspot">adoptopenjdk.net</a>.</li>
<li> Open the file that you downloaded (with Java). If done correctly, the installer window should appear.</li>
<li> Click "OK".</li>
<li> Open the Minecraft launcher (closing it first if it was already open).</li>
<li> A new Forge installation profile should appear. Select it, put the mods you want to use into a "mods" folder in your Minecraft directory (instructions on how to find it <a rel="nofollow" target="_blank" class="external text" href="https://minecraft.fandom.com/wiki/.minecraft">here</a>), and you're ready to play!</li></ol>
<p>A word of safety: <span style="color:#dd2423"><b>NEVER</b></span> download mods or mod loaders from websites you don't trust! At best, this can lead to installing malware and viruses; at worst, you can lose your Minecraft account. For your safety (and everyone else's), only download mods from the websites listed below unless you are sure you know what you're doing.
</p>
<h2><span class="mw-headline" id="OptiFine">OptiFine</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h2>
<p>OptiFine is probably the most popular and most downloaded mod for Minecraft. It's been around since Minecraft Beta 1.7.3, and was originally a great mod with many optimisations, features, and quality of life improvements. However, in recent years, it's benefits have dwindled and causes many incompatibilities with other mods due to it replacing large amounts of code and being closed-source. In the modern Minecraft modding era, with the Fabric mod loader and community, open-source mods exist to replace most of OptiFine's features. Other mods that optimise the game more than OptiFine exist too, and they work in a way that's more compatible with other mods.
</p><p>It's not recommended to use OptiFine anymore, but it is allowed <span style="color:#dd2423"><b>as long as Options -&gt; Video Settings -&gt; Performance -&gt; Fast Math is set to OFF</b></span> (reason why <a rel="nofollow" target="_blank" class="external text" href="https://github.com/sp614x/optifine/issues/5583">here</a>). <a rel="nofollow" target="_blank" class="external text" href="https://www.curseforge.com/minecraft/mc-mods/optifabric">OptiFabric</a> is a mod which allows you to use OptiFine on the Fabric mod loader (it isn't supported natively), and is also allowed, though not recommended.
</p>
<h3><span class="mw-headline" id="How_to_Download_OptiFine_Safely">How to Download OptiFine Safely</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h3>
<ol><li> Go to the download page on <a rel="nofollow" target="_blank" class="external text" href="https://optifine.net/downloads">optifine.net</a>.</li>
<li> Find the version you want to download in the versions list.</li>
<li> Right click the "(mirror)" button link to the right of the "Download" button for the version you want to download.</li>
<li> A context menu should have appeared. In it should be an option called something like "Open Link in New Tab". Click it.</li>
<li> You should now be on the page with the download link. Click it. There are ads here, so be very careful to click on the right link.</li>
<li> You should get a file called something like `OptiFine_&lt;minecraft_version&gt;_HD_U_&lt;letter&gt;&lt;number&gt;.jar`.</li></ol>
<h3><span class="mw-headline" id="How_to_Install_OptiFine_Standalone">How to Install OptiFine Standalone</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h3>
<ol><li> Open the installer .jar file. To do this you'll need to install Java first, which you can do from <a rel="nofollow" target="_blank" class="external text" href="https://adoptopenjdk.net/?variant=openjdk16&amp;jvmVariant=hotspot">adoptopenjdk.net</a>.</li>
<li> Click "Install".</li>
<li> Open the Minecraft launcher (closing it first if it was already open).</li>
<li> A new OptiFine installation profile should appear. Select it, and you're ready to play!</li></ol>
<h2><span class="mw-headline" id="Allowed_Mods">Allowed Mods</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h2>
<h3><span class="mw-headline" id="Performance">Performance</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h3>
<p>Despite being a simple-looking block game, Minecraft can actually be quite difficult to run because it is incredibly unoptimised. Luckily, the community have recognised this and created mods to help. These mods don't affect gameplay, and only exist to improve performance and so are okay to use on the server.
</p>
<ul><li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/sodium">Sodium</a> (<a rel="nofollow" target="_blank" class="external text" href="https://github.com/IrisShaders/sodium-fabric">Iris Fork</a>, <a rel="nofollow" target="_blank" class="external text" href="https://github.com/HyperCubeMC/sodium-fabric">Starline Iris Fork</a>) - Rendering engine and client-side optimisation mod, usually even better than OptiFine.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/lithium">Lithium</a> - Server-side optimisation mod, however still providing some benefit when used on a client to join a non-integrated server.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/phosphor">Phosphor</a> - Lighting engine optimisation mod.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/hydrogen">Hydrogen</a> - Memory optimisation mod, particularly effective in highly modded scenarios.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/starlight">Starlight</a> - Lighting engine replacement mod. This provides significantly better performance than Phosphor, but at the cost of potential mod incompatibility.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/ferrite-core">FerriteCore</a> - Memory optimisation mod, generally more effective than Hydrogen and is compatible with Hydrogen.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/lazydfu">LazyDFU</a> - Startup optimisation mod, providing a significant speed-up in the time it takes to start the game by not compiling DataFixerUpper rules until they're actually needed.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/dynamic-fps">Dynamic FPS</a> - Renders Minecraft at 1fps (or not at all) when the Minecraft window isn't focused, as well as improving background performance.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/ebe">Enhanced Block Entities</a> - Improves performance (and slightly visuals) by rendering block entities (such as beds and chests) as normal blocks unless they need to animate (such as when opening a chest).</li></ul>

<p>While OptiFine is not recommended to be used, you can still use most of its features by using a mod from this list that adds the feature you want to use.
</p>
<ul><li> <a rel="nofollow" target="_blank" class="external text" href="https://modrinth.com/mod/iris">Iris</a> (<a rel="nofollow" target="_blank" class="external text" href="https://github.com/HyperCubeMC/Iris">Starline Fork</a>)- A shader mod with support for OptiFine format shader packs. <li> <a rel="nofollow" target="_blank" class="external text" href="https://www.curseforge.com/minecraft/mc-mods/canvas-renderer">Canvas</a> - An alternative shader and optimisation mod, incompatible with Sodium and Iris.</li>

<h2><span class="mw-headline" id="Outdated_Mods">Outdated Mods</span><a class="edit-section-link todo-enjin" style="display:none;">[edit]</a></h2>
<p>Mods here are allowed to be used on Autcraft, but do not work on the version of Minecraft needed to connect to the Autcraft server when this page was last updated. If a mod here has been updated since then, please make a post about it on <a rel="nofollow" target="_blank" class="external text" href="https://www.autcraft.com/forum/m/16338872/viewforum/3071549">the forums</a> so it can be updated. If a mod here is 2 or more major versions behind the Minecraft version Autcraft uses, it will be considered abandoned and removed from this list.
</p>
<ul><li> <a rel="nofollow" target="_blank" class="external text" href="https://www.curseforge.com/minecraft/mc-mods/varied-mob-textures">Varied Mob Textures</a> - A Custom Entity Textures and Random Entities Textures mod that lets resource packs change the textures of mobs depending on information about them. Uses its own format.</li>
<li> <a rel="nofollow" target="_blank" class="external text" href="https://www.curseforge.com/minecraft/mc-mods/pop-out-chat-fabric">Pop Out Chat</a> (<a rel="nofollow" target="_blank" class="external text" href="https://www.curseforge.com/minecraft/mc-mods/pop-out-chat">forge version</a>) - A mod allowing you to pop the Minecraft chat box into another window.</li></ul>
